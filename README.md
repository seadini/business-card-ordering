# Business Card Ordering

This is a simple web project that allows users to order customized business cards. Users can enter their name, title, email, and the desired quantity of business cards. The form validates inputs and displays the order details.

## Features
- HTML for the page structure
- CSS for a clean and responsive design
- JavaScript for form validation and user interaction

## How to Use
1. Open the `index.html` file in a browser.
2. Fill in the form fields: name, title, email, and quantity.
3. Press the submit button to view your order details.
